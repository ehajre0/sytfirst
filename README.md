## 🚀 Schnellstart

### Voraussetzungen
- Node.js >= 14.0
- npm oder yarn
- Git

### Installation

1. Repository klonen:
```bash
git clone https://github.com/ehajre0/sytfirst.git
cd sytfirst
```

2. Abhängigkeiten installieren:
```bash
npm install
```

3. Konfiguration durchführen:
```bash
cp .env.example .env
```

### Verwendung

```bash
npm start
```
